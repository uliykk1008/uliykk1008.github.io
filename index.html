OBJECT ORIENTED PROGRAMMING

Uliya Gumpari | 251410008 | IPK 3,87

1. Pengertian OOP

Object Oriented Programming adalah paradigma pemrograman yang berorientasi pada objek. OOP membantu membangun program yang terstruktur, modular, dan mudah dikembangkan.

2. Konsep Dasar OOP

Konsep utama OOP meliputi Class, Object, Encapsulation, Inheritance, dan Polymorphism. Konsep ini membuat sistem lebih fleksibel dan reusable.

3. Class Sederhana (Car)

Class merupakan blueprint untuk membuat objek. Contoh class Car dengan atribut dan method.

4. Object Instantiation

Object dibuat dari class menggunakan constructor. Proses ini disebut instantiation.

5. Constructor

Constructor digunakan untuk menginisialisasi nilai awal suatu objek ketika dibuat.

6. Practice Class Mahasiswa

Implementasi class Mahasiswa dengan atribut dan method untuk menampilkan data mahasiswa.

7. Encapsulation

Encapsulation menyembunyikan data dengan membuat atribut private dan mengaksesnya melalui getter/setter.

8. This Keyword

Keyword this digunakan untuk merujuk pada atribut milik class agar tidak terjadi kesalahan referensi variabel.

9. Product Class

Contoh penggunaan final dan named parameter untuk membuat class lebih aman dan fleksibel.

10. Kesimpulan

OOP mempermudah pengembangan perangkat lunak dengan struktur yang jelas melalui penggunaan class dan object. Konsep encapsulation, constructor, dan this keyword meningkatkan keamanan serta keteraturan kode.

11. Quiz Bonus

Buatlah class Buku dengan atribut judul dan penulis serta method tampilkan().
